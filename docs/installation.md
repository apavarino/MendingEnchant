## MendingEnchant - Installation

Make sure you have [downloaded](download.md) the plugin first.

### Install steps

1. Stop your server.
2. Put `mending-enchant-<version>.jar` into your `plugins/` folder.
3. Start the server.
4. Confirm startup in console logs.

Expected log example:

```text
[MendingEnchant] is enabled !
```

### First checks

- Confirm no startup errors are shown.
- Verify `plugins/MendingEnchant/` was generated.
- Check that `config.yml` and `localization/en_US.json` exist.
- Run `/mendingenchant info` in-game with the correct permission to confirm command registration.

If you encounter issues, check [troubleshooting](troubleshooting.md).
