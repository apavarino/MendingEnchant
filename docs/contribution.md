## MendingEnchant - Contribution

Contributions are welcome.

### Code contributions

If you want to propose improvements:

1. Fork the repository.
2. Create a feature or fix branch.
3. Open a pull request with a clear description of the behavior change and any compatibility impact.

Repository: [apavarino/MendingEnchant](https://github.com/stellionix/MendingEnchant)

### Reporting issues

When opening an issue, include:

- server software and version
- Java version
- MendingEnchant version
- relevant config section
- logs or stack traces
