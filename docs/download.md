## MendingEnchant - Download

You can download MendingEnchant from the official sources below:

- [CurseForge](https://www.curseforge.com/minecraft/bukkit-plugins/mendingenchant)
- [BukkitDev](https://dev.bukkit.org/projects/mendingenchant)
- [GitHub](https://github.com/stellionix/MendingEnchant) (build from source)

!!! warning
    Avoid downloading the plugin from unofficial mirrors.

### Requirements

- Minecraft server version: **1.13+**
- Java runtime: **Java 8+**

Supported server implementations include:

- Bukkit
- Spigot
- Paper
- Most Bukkit/Spigot derivatives
